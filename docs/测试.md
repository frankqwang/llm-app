# 测试

两层测试，成本和覆盖范围差异很大。

## 单元测试

无需真机的算法路径。

```powershell
cd C:\dev\llm-app\android
.\gradlew.bat :app:testDebugUnitTest
```

46 个用例覆盖 JSON 提取（LLM 输出热路径）、事件评分、迭代规划器、用户自选等。运行很快（约 15 秒），每次构建都会执行。

## UI 测试

Maestro 流程驱动实际的 Compose UI，在连接的模拟器或真机上运行。详见 [.maestro/README.md](../.maestro/README.md) 了解安装和运行方法。流程覆盖 IA shell、标签路由和聊天回退保障。

流水线端到端路径（加载 Gemma、运行模型、产出 mp4）对典型 CI 预算来说太长——请在真机上每个发布版本手动验证一次。

```bash
maestro test .maestro/flows/smoke.yaml          # 10 秒，验证 4 个标签页正常渲染
maestro test .maestro/flows/chat-fallback.yaml  # 15 秒，验证"无静默失败"回复
```
