# 用户手册

应用启动后直接进入 VlogCopilot 页面（而非 gallery 首页）。Stories 标签页的主卡片有两个操作入口。

---

## 自动模式

1. 点击 **开始挑故事**。如未授权，会提示授予 `READ_MEDIA_IMAGES` / `READ_MEDIA_VIDEO` 权限。
2. AI 扫描相册，分割事件，运行 EventScout，并推荐最佳故事。点击任意故事可查看详情、固定或排除。
3. 点击 **开始制作** 生成。在实时进程面板中可查看进度：下载 → 导入 → 感知 → 标注 → Browse / Audience / Director / Editor / Critic / 渲染 → 完成。
4. MP4 会进入 Videos 标签页。点击任意卡片即可内联播放。可随时点击 **取消** 终止。

---

## 自选模式（v5.0+）

1. 在 Stories 主卡片上点击 **或者，我自己挑素材**。
2. 输入你想要的效果（可选——留空提交会弹出引导对话框，附带示例芯片）。例如"30秒怀旧风的成长vlog"或"做条欢快有节奏的"。
3. 在网格中点击缩略图多选素材（至少 3 张）。
4. 点击 **开始制作**。AI 将你的文本解析为结构化的 UserBrief（节奏/色调/调色/必须出现的主体/字幕策略/时长），构建一个合成事件，并运行相同的 Browser→Audience→Director→Editor→Critic→Render 链路，在每个交接点应用你的覆盖参数。
5. 结果会出现在 Videos 标签页，带有"你做的"徽章。

---

## 迭代模式（v5.0+）

1. 在 Videos 标签页的任意视频卡片上点击 **改一改**。
2. 底部面板以 3 列故事板形式展示当前时间线。点击单元格标记它（出现 2dp 边框），或在文本框中输入反馈，或点击 4 个快捷芯片：**更快** / **更慢** / **去字幕** / **换调色**。
3. 提交。Worker 根据你的输入分派：
   - 仅芯片 **去字幕** / **换调色** → `RENDER_ONLY` 范围，约 30 秒，无 LLM 调用
   - 仅芯片 **更快** / **更慢** → `GLOBAL` 范围，约 2-3 分钟（以新节奏重新挑选镜头）
   - 点击单元格 + 输入文本 → `SHOT_LEVEL` 范围，约 30-60 秒（替换该镜头）
   - 仅文本无单元格选择 → `GLOBAL`，约 2-3 分钟
   - 芯片 + 文本/单元格 → `MIXED`，约 3-4 分钟
4. 进度条会出现在视频列表上方（非阻塞——可继续浏览）。完成后新 MP4 自动播放。
5. 点击卡片右上角的 **History** 图标可对比上一版本。

如果 Worker 在迭代中途被终止（vivo BTM、OOM、强制停止），`_pending_iteration/<eventId>.json` 中的暂存反馈会在下次冷启动时自动恢复（每会话一次尝试——失败的迭代不会循环重试）。
